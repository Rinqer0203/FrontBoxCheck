# FrontBoxCheck
BoxCollider2Dと相性のいいRayを用いたフロント判定スクリプト。  
Physics2D.BoxCastNonAllocを使用しているのでGCを発生させない。  
コンテキストメニューからBoxCollider2Dのサイズに合わせることができる。  
判定するオブジェクトはタグとレイヤーでフィルタリングする。
